# Introduction

1ui is a web3-enabled [React](https://react.dev/) component library by Intuition Systems.

## Usage

```bash
pnpm install @intuition/1ui
```

```bash
npm install @intuition/1ui
```

```bash
yarn add @intuition/1ui
```
